MyFlix API

The MyFlix API allows users to discover information on movies, directors, and genres. Users can sign up, update personal details, and build a list of their favorite movies. It is designed following the principles of REST structure. It features URLs, provides responses in JSON format, and adheres to standard HTTP response codes.
