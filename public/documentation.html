MyFlix API

The MyFlix API allows users to discover information on movies, directors, and genres. Users can sign up, update personal details, and build a list of their favorite movies. It is designed following the principles of REST structure. It features URLs, provides responses in JSON format, and adheres to standard HTTP response codes.


Request URL HTTP Method Request Body Data Format Response Body Data Format
Return list of movies /movies GET N/A An array of JSON objects containing information about every movie
Return list of movies by director /movies/directors/:directorName GET N/A An array of JSON objects containing information about every movie requested by the director
Return list of movies by genre /movies/genres/:genreName GET N/A An array of JSON objects containing information about every movie requested by the genre
Return a movies by title /movies/:title GET N/A An array of JSON objects containing information about the movie
Return list of users /users GET N/A A JSON object containing information about all users, including an assigned ID
Return a user by username /users/:Username GET N/A A JSON object containing information about a specific user, including an assigned ID
Add a user /users POST Enter a JSON object of a new user
example:
"Username": "bobdylan"
"Password": "tangledupinblue"
"Email": "bobdylan@gmail.com"
"Birthday": "1999-08-12" 		A JSON object containing information about an added user, including an assigned ID
Update a user /users/:Username PUT Update a JSON object of a current user
example:
"Username": "newusername"
"Password": "newpassword"
"Email": "newemail@gmail.com"
"Birthday": "new birthday"		 A JSON object containing updated information about an updated user
Delete a user /users/:Username DELETE N/A Response text saying "This user has been deleted"
Add a movie to a user's favorites /users/:Username/movies/:MovieID PUT Select a movie to add to favorites Response indicating whether the movie was successfully added to the user's favorites
Remove a movie from a user's favorites /users/:Username/movies/:MovieID DELETE Unselect movie from favorites Response indicating whether the movie was successfully removed from the user's favorites
